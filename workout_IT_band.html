Weekly Plan (Run + Knee + IT Band Syndrome Focus + Lower Back)

ITBS Focus: Emphasis on single-leg control and reduced TFL dominance.
If an exercise causes next-day hip/TFL soreness, reduce volume or remove it.

Weekly Rules Important

  • No high-volume lateral band work before runs.
  • Strength should not cause lingering soreness.
  • Avoid aggressive IT band rolling near the knee.
DayActivityDetails
Sunday Run Run as planned.
Monday Run + Knee + ITBS Control + Lower Back Warm-Up:
  • Glute Bridges – 2×10
  • Leg Swings (front/back) – 10/side
ITBS / Knee Strength:
  • Single-Leg Squat to Box (Single-Leg Box Squat) – 3×6–8/side (slow 3–4 sec descent)
  • Step-Downs (Forward Step Down) – 3×6/side (use a 6–8" step)
  • Terminal Knee Extensions (TKE) – 3×15/leg
Lower Back:
  • Banded Pull-Through – 3×10–12
Tuesday Rest Optional light mobility or walking.
Wednesday Run Run as planned.
Thursday Adductors & Hamstrings + ITBS Support Strength:
  • Copenhagen Side Plank (Modified) – 2×10–15s/side
  • Romanian Deadlift – 3×10
  • Stability Ball Hamstring Curl (Eccentric) – 3×8
  • Side Plank (Knees Bent) – 2×20–30s/side
Friday Run Run as planned.
Saturday Glute Medius & ITBS Control Strength & Control:
  • Hip Airplanes (Supported) – 3×5/side
  • Single-Leg RDL – 3×6–8/side
  • Side Plank (Knees Bent) – 2×20–30s/side
Progression Rule: Progress only when you can complete a full week of running with no IT band symptoms. Increase one variable at a time (reps, load, or step height).
